# Current Working Hypothesis

Ω appears to behave as a morphology-sensitive structural perturbation measure rather than a semantic correctness metric.

Observed sensitivity:

- control-flow deformation
- AST restructuring
- representational compression and expansion
- local structural transformation intensity

Observed weak or inconsistent sensitivity:

- repository topology
- historical co-change
- semantic equivalence
- static centrality

---

# Interpretation Boundary

OMNIA does not currently demonstrate semantic understanding.

Current evidence suggests that Ω reacts primarily to structural and representational deformation patterns observable in code morphology.

This repository does not claim bug detection, correctness verification, or semantic equivalence proof capability.

# Main Experimental Direction

The current research direction investigates whether structural perturbation behaves as an emergent measurable property independent from:

- code size
- repository history
- dependency topology
- semantic intent

The strongest observed signals currently emerge from:

- AST deformation
- control-flow restructuring
- representational compression and expansion

---

# Important Limitation

Correlation does not imply causation.

Observed relationships may still be partially explained by hidden structural proxies or measurement coupling effects.

Further independent validation is required.
